NOC Mini Widget

Ultra-Enhanced Pack v3.0.0

🚀 Ultra-Enhanced Features

The Ultra-Enhanced Pack adds three powerful visual effects to the NOC Mini Widget:

  • TVL Gradient Trails: Smooth motion trails with afterglow effect during TVL updates
  • Ultra Neon Pulse: Stronger, brighter neon-style pulses for READY/ACTIVE states with outer glow rings
  • Spectral Radar Ring: Animated radar/ring effect for Black Hole Mode ENGAGED status

All effects respect prefers-reduced-motion and are disabled when the widget is collapsed for optimal performance.

⚙️ Configuration

Enable the Ultra-Enhanced Pack by setting configuration before loading the widget:

window.NOC_MINI_WIDGET_CONFIG = {
  // Enable Ultra-Enhanced Pack
  ultraEnhanced: true,
  
  // Individual effect toggles (all default to true)
  ultraTvlTrails: true,      // TVL gradient trails
  ultraNeonPulse: true,       // Stronger neon pulses
  ultraRadarRing: true,       // Black Hole radar ring
  
  // Standard options still available
  enableChart: true,
  chartMaxHistory: 20,
  tvlAnimationSpeed: 0.1
};

📊 How to Observe Effects

  • TVL Trails: Click "Start Live Simulation" and watch the TVL row - you'll see gradient motion trails as values change
  • Neon Pulse: When Omni-Mesh shows "ACTIVE" or AI Self-Healing shows "READY", notice the stronger pulsing glow with outer rings
  • Radar Ring: When Black Hole Mode shows "ENGAGED", a rotating radar ring appears with concentric circles and a sweeping beam

📝 Usage Instructions

To integrate the Ultra-Enhanced Pack into your application:

  1. Set window.NOC_MINI_WIDGET_CONFIG with ultraEnhanced: true
  2. Include noc-mini-widget.css and noc-mini-widget.js
  3. Add the widget HTML structure to your page
  4. Update metrics using window.updateNOCWidget(data)
